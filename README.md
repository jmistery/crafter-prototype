# 五岳CMS-lite 高保真HTML原型

## 📋 项目概述

本项目包含基于《五岳CMS-lite产品需求文档(PRD)V1.0》的P0功能高保真HTML原型，使用Element UI组件库实现，提供完整的交互式演示。

## 🎯 原型列表

### P0-01: Header 导航组件
**文件:** `P0-01-header-navigation.html`

**功能特性:**
- ✅ 可配置的企业Logo展示
- ✅ 多级导航菜单（支持二级下拉）
- ✅ 中英文语言切换
- ✅ 全局搜索入口
- ✅ 用户中心入口和登出功能
- ✅ 响应式设计（桌面端/平板端/移动端）
- ✅ 移动端汉堡菜单和侧边抽屉

### P0-02: Homepage Banner
**文件:** `P0-02-homepage-banner.html`

**功能特性:**
- ✅ 三种Banner类型（图片/视频/轮播）
- ✅ 可配置标题、副标题和CTA按钮
- ✅ 实时配置面板
- ✅ 行业预设模板（物流/零售）
- ✅ 响应式高度和布局
- ✅ 动画效果和滚动提示

### P0-12: 白标化配置中心
**文件:** `P0-12-whitelabel-config.html`

**功能特性:**
- ✅ Logo上传和管理（主Logo/暗色Logo）
- ✅ 品牌色系配置（主色/辅助色/功能色）
- ✅ 智能色彩选择器和推荐色板
- ✅ 字体系统设置（标题字体/正文字体）
- ✅ 四套行业预设模板
- ✅ 实时预览（桌面端/平板端/移动端）
- ✅ 配置导出功能

### P0-13: 资产库管理
**文件:** `P0-13-asset-reference.html`

**功能特性:**
- ✅ 资产上传和管理
- ✅ 网格视图/列表视图切换
- ✅ 地址化引用机制（标准URL/CDN URL）
- ✅ 版本历史和回滚
- ✅ 引用页面追踪
- ✅ 批量操作（添加标签/下载/删除）
- ✅ 存储统计和资产分类
- ✅ 筛选和搜索功能

## 🚀 快速开始

### 方式一：直接打开索引页面
1. 打开 `index.html` 文件
2. 点击相应的原型卡片查看各个功能

### 方式二：直接打开单个原型
直接在浏览器中打开任意HTML文件即可查看对应原型

### 方式三：使用本地服务器（推荐）
```bash
# 使用Python启动本地服务器
python3 -m http.server 8000

# 或使用Node.js的http-server
npx http-server -p 8000

# 然后在浏览器中访问
http://localhost:8000/
```

## 🛠️ 技术栈

- **HTML5**: 语义化标签
- **CSS3**: Flexbox、Grid、动画、响应式设计
- **Vue.js 2.6**: 数据绑定和组件交互
- **Element UI**: UI组件库
- **CDN**: 所有依赖通过CDN引入，无需npm安装

## 📱 响应式设计

所有原型都支持多种设备尺寸：
- 🖥️ **桌面端**: ≥1200px
- 📱 **平板端**: 768px - 1199px
- 📱 **移动端**: <768px

## 🎨 设计规范

### 颜色系统
- **主色调**: #0066CC (蓝色 - 物流行业)
- **辅助色**: #00CC66 (绿色)
- **成功色**: #67C23A
- **警告色**: #E6A23C
- **错误色**: #F56C6C

### 字体系统
- **标题字体**: Roboto, Microsoft YaHei
- **正文字体**: Open Sans, PingFang SC

### 间距规范
- 基础间距: 8px的倍数（8px, 16px, 20px, 24px, 40px）

## 📖 功能说明

### P0-01 交互功能
- 点击Logo返回首页
- 鼠标悬停显示下拉菜单
- 点击语言切换图标切换中英文
- 点击搜索图标打开搜索对话框
- 点击用户头像显示下拉菜单
- 移动端点击汉堡菜单打开侧边抽屉

### P0-02 配置功能
- 切换Banner类型（图片/视频/轮播）
- 调整Banner高度和遮罩透明度
- 编辑标题、副标题和按钮文本
- 应用行业预设模板
- 实时预览效果

### P0-12 配置功能
- 上传主Logo和暗色Logo
- 使用色彩选择器或推荐色板选择品牌色
- 选择标题和正文字体
- 应用四套行业预设模板
- 实时预览在不同设备上的效果
- 导出配置为JSON文件

### P0-13 管理功能
- 上传图片、视频、文档等资产
- 网格视图和列表视图切换
- 筛选资产类型、标签和日期
- 查看资产详情和版本历史
- 复制资产引用链接
- 批量操作选中的资产
